# لوحة تحليل استهلاكات فروع قطر — مايو 2026

لوحة تحكم تفاعلية (Dashboard) لتحليل استهلاكات وهوالك ودقّة جرد فروع **لبن قطر** الأربعة:
**الغرافة · النصر · الوكرة · الفروسية** — لشهر مايو 2026.

**إعداد:** بلال الأنصاري — إدارة تشغيل قطر
**مُقدَّم إلى:** المدير العام / أ. خالد محمد

---

## المحتويات (Features)

- 🌓 تبديل الوضع الغامق / الفاتح (Dark / Light mode).
- 📊 أقسام: نظرة عامة شاملة · الفروع (مع شريط اختيار الفرع) · إجمالي الفروع · المقارنات · التحليلات · التوصيات والخطة.
- 📈 رسوم بيانية تفاعلية كثيرة (Bar / Doughnut / Pie / Radar / Bubble / Line / Stacked).
- 🎯 مؤشرات أداء (KPIs) وجداول مقارنة مع ترتيب الفروع.
- ✍️ عنوان وتوقيع مضيء ثلاثي الأبعاد بخط عربي مميّز، وإطار دائري مضيء للّوجو.

---

## الملفات

| الملف | الوصف |
|------|------|
| `index.html` | الصفحة الرئيسية للوحة التحكم |
| `app.js` | منطق العرض وكل الرسوم البيانية |
| `data.js` | البيانات المستخرجة من شيتات الجرد (مدمجة) |
| `data.json` | نفس البيانات بصيغة JSON (مرجعية) |
| `chart.umd.js` | مكتبة Chart.js (محلية — لا تحتاج إنترنت) |
| `vercel.json` | إعدادات النشر على Vercel |
| `preview.html` | نسخة معاينة ذاتية الاكتفاء (كل شيء في ملف واحد) |

## وضع لوجو الشركة (طريقتان)

**الطريقة الأسهل (من داخل اللوحة):** اضغط زر **«رفع اللوجو»** أعلى يسار الشاشة واختر صورة — ستظهر فوراً داخل الإطارين الدائريين المضيئين.

**الطريقة الدائمة (للنشر):** ضع ملف صورة باسم `logo.png` بجوار `index.html` داخل المشروع — ستُحمَّل تلقائياً عند فتح اللوحة دون أي ضغط.
> يُفضّل لوجو مربّع (مثلاً 400×400) بخلفية شفافة (PNG) ليبدو مثالياً داخل الإطار الدائري.

## النشر على GitHub ثم Vercel

### 1) الرفع على GitHub
```bash
# داخل مجلد المشروع
git init
git add .
git commit -m "Qatar branches consumption dashboard — May 2026"
git branch -M main
git remote add origin https://github.com/<اسم-حسابك>/qatar-branches-dashboard.git
git push -u origin main
```

### 2) النشر على Vercel
1. ادخل إلى [vercel.com](https://vercel.com) وسجّل الدخول بحساب GitHub.
2. اضغط **Add New… → Project**.
3. اختر مستودع `qatar-branches-dashboard`.
4. اترك كل الإعدادات كما هي (Framework Preset = **Other**، لا حاجة لأمر بناء).
5. اضغط **Deploy** — سيظهر الرابط خلال ثوانٍ.

> اللوحة عبارة عن موقع ثابت (Static) بالكامل ولا تحتاج أي خادم أو متغيرات بيئة.

### بديل سريع بدون GitHub (Vercel CLI)
```bash
npm i -g vercel
cd qatar-dashboard
vercel        # للمعاينة
vercel --prod # للنشر النهائي
```

## ملاحظة منهجية حول البيانات
تختلط وحدات القياس بين (كجم · لتر · عدد). تجميع «الوحدات» يُستخدم للاتجاه العام،
بينما تُحسب **نِسَب الهالك على الخامات فقط (كجم/لتر)** لضمان الدقّة، وتظهر تفاصيل كل خامة
في جدول كل فرع. الأرقام مستخرجة مباشرة من صفحة `total` في كل شيت جرد شهري.
